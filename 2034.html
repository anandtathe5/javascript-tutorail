JavaScript Topics Tutorial

1. Functions

JavaScript functions are blocks of code designed to perform a particular task. They are executed when something invokes them.

                function displayMessage() {
                    document.getElementById('message').innerText = "Hello, this is a JavaScript function!";
                }

2. Strings I

Strings in JavaScript are used for storing and manipulating text.

                function greet() {
                    const name = document.getElementById('nameInput').value;
                    document.getElementById('greeting').innerText = `Hello, ${name}! Welcome to JavaScript Strings.`;
                }

3. Numbers

JavaScript numbers can be written with or without decimals. Click the button to see an example of basic arithmetic operations:

                function showNumbers() {
                    const num1 = 5;
                    const num2 = 10;
                    const sum = num1 + num2;
                    document.getElementById('numberResult').innerText = `5 + 10 = ${sum}`;

4. Variables

JavaScript variables are containers for storing data values. Click the button to see how variables are declared and used:

                function showVariables() {

5. Conditions

JavaScript conditional statements are used to perform different actions based on different conditions.

                function checkAge() {
                    let age = prompt("Enter your age:");
                    if (age >= 18) {
                        document.getElementById('ageCheck').innerText = "You are an adult.";
                    } else {
                        document.getElementById('ageCheck').innerText = "You are a minor.";
                    }
                }

6. Arrays I

Arrays in JavaScript are used to store multiple values in a single variable.

                    function displayArray() {
                    const fruits = ["Apple", "Banana", "Cherry", "Date", "Elderberry"];
                    const list = document.getElementById('fruitList');
                    list.innerHTML = '';
                    fruits.forEach(fruit => {
                        const li = document.createElement('li');
                        li.innerText = fruit;
                        list.appendChild(li);
                    });
                }

    7. Arrays II & Callbacks

        13. Array Reduce

        Using the reduce method to accumulate array values.

                        function showReducedValue() {

            29. Class Concepts

            Understanding concepts such as inheritance, encapsulation, and polymorphism in classes.

                            class Vehicle {
                    constructor(make, model) {
                        this.make = make;
                        this.model = model;
                    }
                }

                class Car extends Vehicle {
                    constructor(make, model, doors) {
                        super(make, model);
                        this.doors = doors;
                    }

                    describe() {
                        return `${this.make} ${this.model} with ${this.doors} doors`;
                    }
                }

                function showClassConcepts() {
                    const car = new Car("Toyota", "Corolla", 4);
                    document.getElementById('classConceptsResult').innerText = car.describe();
                }

            30. Class Inheritance

            Understanding inheritance in JavaScript classes.

                            class Employee {
                    constructor(name, position) {
                        this.name = name;
                        this.position = position;
                    }
                }

                class Manager extends Employee {
                    constructor(name, position, department) {
                        super(name, position);
                        this.department = department;
                    }

                    describe() {
                        return `${this.name} is a ${this.position} in ${this.department} department`;
                    }
                }

                function showClassInheritance() {
                    const manager = new Manager("John", "Manager", "Sales");
                    document.getElementById('classInheritanceResult').innerText = manager.describe();
                }

            31. Prototypal Inheritance

            33. Public and Private Fields

            Using public and private fields in JavaScript classes.

                            class BankAccount {
                    #balance = 0; // Private field

                    constructor(owner) {
                        this.owner = owner; // Public field
                    }

                    deposit(amount) {
                        this.#balance += amount;
                    }

                    getBalance() {
                        return this.#balance;
                    }
                }

                function showPublicPrivateFields() {
                    const account = new BankAccount("Alice");
                    account.deposit(100);
                    document.getElementById('publicPrivateFieldsResult').innerText = `Owner: ${account.owner}, Balance: ${account.getBalance()}`;
                }
